# Mongo API setup

## Environment init

### MongoDB
- Tested with v. 7.0.6
- Connection: `mongodb://localhost:27017`
- Database: `links`
- Collection: `links`

### Python:
- Tested with v. 3.11.1
- `> pip install flask pymongo unittest`
- For running check the local Python env settings

## Start Server
- `> python routes.py`

## Run Tests
- `> python TEST_linkManager.py`
